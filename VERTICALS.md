# Verticals

This platform should eventually support many local service-business verticals. Cutler is the landscaping / property-services vertical and should remain the first working template.

## Likely future verticals

| Vertical | Likely tools needed |
| --- | --- |
| Lawn care | Recurring maintenance plans, route/job scheduling, lawn-size calculators, before/after photos, seasonal services. |
| Landscaping | Design/project requests, plant/material catalog, AI yard concepts, square-footage calculators, estimates, project carts. |
| Handyman | Repair requests, punch lists, hourly/project estimates, worker notes, photo proof, property maintenance history. |
| Cleaning | Residential/commercial checklists, recurring billing, room counts, supply notes, before/after photos, quality review. |
| Pressure washing | Surface calculators, before/after photos, water access notes, stain/damage notes, driveway/siding/deck estimate templates. |
| HVAC | Service calls, equipment records, maintenance plans, emergency dispatch, parts tracking, technician notes. |
| Painting | Room/wall calculators, color selections, prep checklists, material estimates, progress photos, approvals. |
| Mobile mechanic | Vehicle profiles, diagnostic requests, parts tracking, mobile appointment scheduling, repair proof, invoices. |
| Pest control | Treatment plans, property notes, safety disclosures, recurring service, pest activity logs, compliance records. |
| Property preservation | Work orders, photo proof, occupancy notes, debris/removal tasks, recurring inspections, vendor/crew tracking. |
| Pool maintenance | Recurring routes, chemical readings, equipment notes, photo proof, repair requests, seasonal openings/closings. |
| Junk removal | Volume/load estimates, photo-based quoting, disposal fees, crew scheduling, before/after proof. |
| Moving services | Inventory lists, crew scheduling, truck needs, mileage/time estimates, damage documentation, payment collection. |
| Small contractor services | Estimates, change orders, project milestones, materials, invoices, proof photos, customer approvals. |
| Senior living maintenance services | Recurring maintenance, safety checklists, family/customer communication, worker proof, property history. |

## Shared pattern

Each vertical should reuse the same core platform while supplying its own services, forms, calculators, statuses, checklists, proof requirements, pages, and admin views.
