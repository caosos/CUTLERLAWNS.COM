# Admin / Owner Dashboard

The admin and owner dashboard should eventually allow authorized users to:

- view all customers;
- view all properties;
- view service requests;
- create and approve estimates;
- create invoices;
- track paid/unpaid invoices;
- manage recurring maintenance schedules;
- assign jobs to workers;
- review worker-uploaded proof photos;
- view customer complaints/issues;
- see job history by property;
- manage products/plants/services;
- manage gallery images;
- manage public website content;
- manage AI yard design requests;
- view basic P&L/business reporting;
- export reports;
- control user roles and permissions.

Owners/superadmins should see all operational data. Admin actions should be auditable where practical.
