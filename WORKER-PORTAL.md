# Worker / Crew Portal

Workers or crews should eventually be able to:

- log in;
- see assigned jobs;
- see property notes needed to perform the job;
- see service checklists;
- upload before photos;
- upload after photos;
- mark work complete;
- add internal job notes;
- report issues found on-site;
- document weather/access problems;
- record materials used;
- record time or job duration;
- submit completion proof.

## Accountability purpose

If a customer disputes work, the owner should be able to review assigned crew, date/time, before photos, after photos, checklist, worker notes, materials used, and completion timestamp.

Workers should not see financial reports unless explicitly allowed by role permissions.
